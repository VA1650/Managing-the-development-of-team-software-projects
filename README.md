# Document-Constructor-Engine

Автоматизированная система управления шаблонами документов для коммерческой организации. Проект предназначен для быстрой генерации юридических и кадровых документов на основе динамических шаблонов.
### 🛠 Мой вклад (Backend-разработка)

В рамках командной разработки я отвечала за серверную часть и архитектуру данных:

    Проектирование БД (SQLite): Разработка структуры таблиц для хранения профилей партнеров, шаблонов документов и метаданных пользователей.

    Логика автозаполнения: Реализация движка, который автоматически парсит шаблоны и подставляет переменные (ФИО, даты, оклады, должности и т.д.) из базы данных.

    Система шаблонизации: Создание бэкенд-логики для управления кастомными шаблонами от разных партнеров.

    API/Business Logic: Обработка запросов на генерацию готовых к подписи документов.

### 🚀 Ключевые функции

    Многопользовательские шаблоны: Поддержка уникальных форм документов для разных организаций-партнеров.

    Динамические поля: Автоматическая генерация документов с учетом актуальных кадровых данных.

    Конструктор: Возможность добавления новых типов документов в систему без изменения исходного кода.

### 📈 Технический стек

    Core: Python

    Storage: SQLite

    Logic: Модульная архитектура для легкой интеграции новых партнеров.
